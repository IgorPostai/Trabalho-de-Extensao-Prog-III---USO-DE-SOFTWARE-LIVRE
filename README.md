# 🐧 Uso de Software Livre

## 📚 Projeto de Extensão – Programação III

Este projeto tem como objetivo conscientizar sobre a importância do Software Livre, apresentando alternativas gratuitas, seguras e colaborativas para uso no dia a dia.

---

# O que é Software Livre?

Software Livre é aquele que garante aos usuários as quatro liberdades fundamentais:

✅ Executar o programa para qualquer finalidade

✅ Estudar como ele funciona

✅ Redistribuir cópias

✅ Modificar e compartilhar melhorias
